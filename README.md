# Image-or-Text-Recognition
AI Project 4 - Text Recognition using OCR

# Objective:
To extract text from an image using Optical Character Recognition (OCR).

# Tools Used:
1. Python
2. OpenCV
3. Tesseract OCR

# Procedure:
1. Read the image.
2. Convert image into grayscale.
3. Apply Gaussian Blur.
4. Extract text using Tesseract OCR.
5. Display the extracted text.
